== Description ==

wpForo is one of the popular forum plugins. Using forums with your courses will increase the engagement of your learners with your courses.

With wpForo, you have an inbuilt option for whom your forums will be visible.

You can select whether a user must be enrolled in ALL of the associated courses versus ANY of the associated courses.

If you select ‘ALL’ then the learner has to be enrolled in all courses before they will be able to access the forum.

If you select ‘ANY’, then the learner needs to only be enrolled in any one of the associated courses in order to access and participate.

You can include a custom message to display access limitation. This field supports HTML markup so if you are selling courses you can include a link to your sales page!

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `learndash-wpforo.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How to link course with forums =

At single forums, you will have options to link with available courses.

= Can we link forum with multiple courses? =

Yes, you have inbuilt option to to link multiple courses.
